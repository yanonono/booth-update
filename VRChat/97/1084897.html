Model description

It is the original 3D model KARAN of Humanoid format created for VRChat. By purchasing this model, you agree to the terms of use. Please read the terms of service carefully and purchase. · Original model KARAN Producer: リリク Number of triangular polygons: 14595 polygon - It is the number of polygons contained in a hood, a knife, and a holster. - The number of polygons varies depending on the daggers and masks included in the model data, with or without costumes. · Lip Sync: 15 species · Eye Tracking: correspondence · BlendShape: 58 species - Includes Lip Sync and Eye Tracking.

Content guide

· Animation: Folder containing Unity facial expression animation produced for VRChat · Material: Folder containing model texture · PSD: A folder containing a psd file that can display a UV grid for texture modification · Dagger.FBX: Dagger, model containing holster · Holster_L, R.FBX: Holster model divided into left and right · KARAN.FBX: KARAN model data - Dagger, Holster, Mask are not included. · Mask.FBX: Model containing mask

how to use

· How to use the included unity package 1) Create a new Unity project in Unity 2) Import VRCSDK and avatar shader. 3) Select the package that matches the imported shader of KARAN _OOO.unitypackage in the downloaded folder and import it into the project. 4) Once the import is completed, open KARAN / KARAN.prefab and upload it. · Mesh description in KARAN.FBX - Body: It is a face mesh. - Hair: It is a hair mesh. - Hand: It is a two-handed mesh. - Hood: It is a costume mesh except for the head part. - Hood_Close: A mesh that wrote a hood on his head. - Hood_Open: It is a mesh that removed the hood. - Shoes: It is a shoe mesh. - Skin_LegNeck: It is a mesh of the body except the part overlapping with the costume. Please activate this mesh when dressing Hood costumes. - Skin_Total: This mesh contains the whole body. Please disable this mesh when dressing Hood costume. v1.1 Addition - Ear: It is an ear mesh. - Hand_Skin: It is a bare hand mesh. · MMS Shader 1.3 : https://twitter.com/Mini_9321_VR/status/1013973807323742208
