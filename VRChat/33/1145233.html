# 概要 VRChatのログファイルから、Joinしたワールドやフレンドの名前を抽出するツールです。 # つかいかた 1. 下記にあるVRChatのログファイルをさがしてください。「?」にはログファイルが作成された日時やAM/PMが入ります。 C:/Users/(username)/AppData/LocalLow/VRChat/vrchat/output_log_??_??_??_??.txt 2. 探したファイルを vrc_world_user_checker_v03.exe にドラッグ＆ドロップします。 3. ログファイルと同じ場所に、VRChat_usrlog.txt が作成されるので見てください。 # 出力例 Joinしたワールドとフレンド名と日時が、VRChat_usrlog.txtの末尾に追記されていきます。 2018.12.19 22:36:07 World: Suna's Playground 2018.12.19 22:36:10 User: sunasaji ... # 注意 VRChatのログファイルは順次消えていきます。 VRCを抜けたときとかに本ツールで情報を抽出しておくとよいと思います。 # 関連情報 下記のブログ記事をみてください。 http://suna.hateblo.jp/entry/2018/12/21/000000 # ソース 下記に公開中です。 https://github.com/sunasaji/VRC_log_checker Python3で書かれています。EXE化するためにPyinstallerを使っています。 # ライセンス CC0なので好きに使ってください。 http://creativecommons.org/publicdomain/zero/1.0/deed.ja # 更新情報 2018/12/20 v0.1 初版公開 2018/12/27 v0.2 AudioTrigger,AnimationBool等の文字列を拾う不具合を修正 2020/04/10 v0.3 VRChat更新に伴いユーザ名の表示フォーマットが変わったことに対応
