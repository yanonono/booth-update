how to use

· How to use the included unity package 1) Create a new Unity project with Unity. 2) Please import DynamicBone if you want VRCSDK and shaking items. * Although DynamicBone is set up, assets need to be prepared separately. 3) Please import RANA.unitypackage in the downloaded folder 4) Once the import is completed, open RANA / RANA.prefab and upload it.
