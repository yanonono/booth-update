Windows -> Shiggie Image tool 2. ステップ1を解除します。 3. ".png / .jpg"を含む場所を入力します。 /Materialsが存在する必要はありません。 4. "Generate Materials"を押します。 *入力したパスの最後に "/"を含める *プログラムをテストするには、 "Maximum Generated Materials"を小さい値に設定します。 *警告**大量のMaterialsには長い時間がかかります。このプロセスは、毎秒約2つのMaterialsです。 https://i.imgur.com/xx4iRHG.png ======================== 2 - アニメーションを生成する ======================== 1.ステップ2を解除します。 - Generate Materialsへのパスが正しいことを確認してください。 2.生成したAnimation Clipの場所を入力します。 3. [Generate .anim]を押します。 *このプロセスは、ハイエンドコンピュータでは多くのMaterialsでも15秒以内で完了します。 https://i.imgur.com/bznGlaz.png https://i.imgur.com/gnwmQtX.png 4.その後、アニメーションクリップをAnimationコントローラに追加してゲームオブジェクトに添付します。 ------------------------- おすすめ ------------------------- - Crunch Texturesクランチテクスチャ https://i.imgur.com/lKbI6wy.png - 低解像度の画像を使用するか、またはFFmpegのコマンドで（ステップ0） - Mesh Renderer / Skinned / Particleを忘れずに使用する予定のオブジェクトに設定してください。 https://i.imgur.com/RPrQQDg.png =================================
